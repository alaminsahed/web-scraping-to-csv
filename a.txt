# web-scraping-to-csv with axios


## Technical Stack
- Node.js
- Express.js
- Cheerio


## About Project:
Fecth data of a movie from IMDB website and it in CSV file.



## How to install

- Download or Fork or Clone this repo
- Run the command
```
npm install
```
- Run the project
```
npm start
```
or
- Run with nodemon
```
npm run dev
```
